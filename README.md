# OcctImguiPort

Occt + ImGui + custom GL render

C# port of https://github.com/eryar/OcctImgui

There are three options to use:
1. (default) OpenTK.GLControl (GLFW) binding
2. GLFW native window
3. WinForm Control (HWND)
